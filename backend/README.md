# GrowEasy CRM Backend

Node/Express application mapping CSV columns and processing CRM data.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and fill in necessary API keys.

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build and start in production:
   ```bash
   npm run build
   npm start
   ```
